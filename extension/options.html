Open in Browser offers the choice to display a file directly in the browser or to download and save the file. This dialog is only activated if the server has marked a file as downloadable, or if Chrome has no default content handler for the given document.

Preferences

Some servers incorrectly serve pages as text. Chromium deals with this issue by checking whether the page is really plain text. An annoying consequence is that when text files contain binary characters, it will be downloaded instead of displayed.
Turn this option on to always view text files in the browser.
